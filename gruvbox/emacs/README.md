<h2 align="center">Deploying</h2>
Move the directory to your `.config` folder and the `.emacs` file in your home dir.
In your `.Xresources` you are required to have a gruvbox theme and then run:

```
xrdb ~/.Xresources
```
like you normally would for X programs.

for Vim Keybindings (Evil Mode), clone the repository in dir `~/.config/emacs.d/`
